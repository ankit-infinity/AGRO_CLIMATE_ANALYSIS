# OVERVIEW

Built an end to end agro climate analytics solution using AWS,snowflake,and power
bi to analyse climate pattern ,support data driven agricultural insights and enable inteactive and scalable
reporting dashboard.

# METHODS:

1.Creating Amazon S3 Bucket and load the data into it .

2.creating the role using IMA.

3.creating the integration object in snowflake using sql command.

4.loading data into snowflake and performing some data transformation and rainfall group column.

5.import data into powerbi from snowflake

6.adding rainfall,temperature,humidity and yield page  in the report

7 publish the report in bi service.
